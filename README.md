# 🏠 Govind Bhawan – IIT Roorkee Portal

A modern, interactive hostel management web interface designed for Govind Bhawan, IIT Roorkee, integrating essential student services into a single seamless platform.

---

## ✨ Features

### 🍽️ Mess Attendance
- Mark present/absent for meals (Breakfast, Lunch, Dinner)
- Track daily & weekly attendance
- Helps optimize food planning

### 🛠️ Complaint Portal
- Raise complaints (Water, Electrical, Internet, etc.)
- Set priority levels
- Track status: Open → In Progress → Resolved

### 🏨 Guest Room Booking
- Book rooms for visitors (Standard, Deluxe, Suite)
- View availability calendar
- Automatic price calculation

### 🏅 Sports Facilities
- Explore available sports & timings
- View equipment and booking info
- Quick request for booking

### 🖼️ Gallery
- Filter by categories: Sports, Cultural, Academic, etc.
- Visual representation of hostel life

### 📢 Notices & Events
- Dynamic notice board
- Monthly calendar with highlighted events
- Upcoming events section

### 🌗 Dark Mode
- Toggle between light and dark themes

---

## 🛠️ Tech Stack

- **HTML5** – Semantic markup
- **CSS3** – Custom design system with variables
- **Vanilla JavaScript** – No frameworks required
- **Responsive Design** – Mobile & Desktop optimized

---

## 🚀 How to Run

1. Download or clone the repository
2. Open the file: `index.html`
3. Run in any browser (Chrome recommended)
